My IMDb page embeds — demo

This GitHub Pages page attempts to show IMDb pages inside iframes. Many sites (including IMDb) often block embedding — if that happens you'll see the fallback (link + poster). Use the edit instructions below to replace the example movies with your own IMDb links.

How to deploy: create a GitHub repo, add this file as index.html, push, then enable GitHub Pages from the repository settings (branch: main / root). The page will be available at https://<your-username>.github.io/<repo-name>/.
